# Hoshikuzu 文档

## 介绍

Hoshikuzu 是一款高性能、高稳定性、易扩展的内容管理系统，采用现代化的技术栈构建，为开发者提供强大的内容管理和业务扩展能力。

### 核心特性

- **高性能**：基于 GoFiber 框架构建，提供出色的并发处理能力和响应速度
- **高稳定性**：完善的错误处理机制和日志系统，确保系统稳定运行
- **易扩展**：模块化设计，支持灵活的功能扩展和定制开发
- **现代化技术栈**：后端采用 Go + GoFiber + Ent ORM，前端采用 Vue3 + TypeScript + Naive UI

### 技术架构

#### 后端技术栈

- **框架**：GoFiber v2.52.9
- **ORM**：Ent v0.14.5
- **日志**：slog / zap 结构化日志
- **API 设计**：RESTful 风格

#### 前端技术栈

- **框架**：Vue3
- **语言**：TypeScript
- **UI 组件库**：Naive UI
- **文档**：Docsify

### 模块概览

Hoshikuzu 提供了丰富的功能模块，满足各种业务场景需求：

#### 内容管理模块

- 文章管理：支持文章的创建、编辑、发布、分类、标签等功能
- 内容审核：完善的内容审核流程
- SEO 优化：内置 SEO 优化功能

#### 支付模块

- 支付宝支付：支持支付宝扫码支付、网页支付、移动支付
- 微信支付：支持微信扫码支付、公众号支付、小程序支付
- 易支付：支持易支付平台接入

#### AI 模块

- AI 摘要：自动生成文章摘要
- AI 播客：将文本内容转换为音频播客
- AI 写作：智能辅助写作功能
- AI 知识库：构建和管理知识库

#### 商城模块

- 商品管理：商品的创建、编辑、上下架、库存管理
- 会员系统：会员等级、积分、权益管理
- 优惠券：优惠券创建、发放、核销

#### 社交模块

- 说说：用户动态发布和互动
- 朋友圈：好友动态分享
- 社交登录：支持第三方社交平台登录
